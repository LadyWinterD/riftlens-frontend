## 🔧 如果遇到问题

### 装备图标不显示？
1. 检查 `public/items/` 目录是否有图标文件
2. 重启开发服务器
3. 强制刷新浏览器（Ctrl+Shift+R）

### 搜索不到玩家？
1. 检查玩家名称拼写
2. 确保大小写正确
3. 只搜索 EUW 服务器的玩家

### 需要更新图标？
```bash
# 删除旧图标
rmdir /s /q public\items

# 重新下载
node scripts/download-item-icons.js
```
