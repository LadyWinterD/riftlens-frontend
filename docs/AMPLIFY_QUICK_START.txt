WHAT YOU NEED:
================================================================

1. AWS Account (logged in)
2. GitHub Repository (code pushed)
3. API Gateway URL (from AWS Lambda)

================================================================

5-STEP DEPLOYMENT:
================================================================

STEP 1: Visit Amplify Console
   >> https://console.aws.amazon.com/amplify/

STEP 2: Create New App
   - Click "New app" -> "Host web app"
   - Select "GitHub"

STEP 3: Connect Repository
   - Authorize AWS Amplify
   - Select your repository and branch

STEP 4: Configure Environment Variables (IMPORTANT!)
   Expand "Advanced settings", add:
   
   +-------------------------------------------------------+
   | NEXT_PUBLIC_API_GATEWAY_URL                           |
   | = https://xxxxx.execute-api.us-east-1.amazonaws.com   |
   |                                                        |
   | NEXT_PUBLIC_CHAT_API_URL                              |
   | = https://xxxxx.execute-api.us-east-1.amazonaws.com   |
   |                                                        |
   | NEXT_PUBLIC_AWS_REGION                                |
   | = us-east-1                                           |
   +-------------------------------------------------------+

STEP 5: Save and Deploy
   - Click "Save and deploy"
   - Wait 3-5 minutes
   - Done!
